# devbox

An Ubuntu 14.04 (Trusty Tahr) machine with the following installed:

  - htop
  - tree
  - unzip
  - zsh
  - tmux
  - git
  - vim
  - ruby 2.1.5 and 2.2.1
  - chruby
  - ruby-install

## Usage

You'll need to have [vagrant](vagrantup.com) installed.

Then you can run:

    vagrant up
    vagrant ssh
